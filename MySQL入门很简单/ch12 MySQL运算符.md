实践

创建并插入数据
```
mysql> show databases;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| first_database     |
| mysql              |
| performance_schema |
| test               |
+--------------------+
5 rows in set (0.37 sec)

mysql> use test;
Database changed

mysql> create table t(
    -> num int,
    -> str varchar(20));
Query OK, 0 rows affected (0.84 sec)

mysql> insert into t values(30, 'mysql');
Query OK, 1 row affected (0.03 sec)
```

算术运算
```
mysql> select num, num+4, num-4, num div 4, num % 4 from t;

| num  | num+4 | num-4 | num div 4 | num % 4 |

|   30 |    34 |    26 |         7 |       2 |

```

t;

|   30 |      0 |       1 |      1 |         1 |        0 |         0 |

```

逻辑运算
```
mysql> select 2&&0, 2&&NULL, 0 and null, 2 || 0, 2 || null, 0 or null;
+------+---------+------------+--------+-----------+-----------+
| 2&&0 | 2&&NULL | 0 and null | 2 || 0 | 2 || null | 0 or null |
+------+---------+------------+--------+-----------+-----------+
|    0 |    NULL |          0 |      1 |         1 |      NULL |
+------+---------+------------+--------+-----------+-----------+
1 row in set (0.00 sec)

mysql> select !2, !0, not null, 2 xor 0, 2 xor null, 0 xor null;
+----+----+----------+---------+------------+------------+
| !2 | !0 | not null | 2 xor 0 | 2 xor null | 0 xor null |
+----+----+----------+---------+------------+------------+
|  0 |  1 |     NULL |       1 |       NULL |       NULL |
+----+----+----------+---------+------------+------------+
1 row in set (0.00 sec)

mysql> select 4&6, 4 | 6, ~4;
+-----+-------+----------------------+
| 4&6 | 4 | 6 | ~4                   |
+-----+-------+----------------------+
|   4 |     6 | 18446744073709551611 |
+-----+-------+----------------------+
1 row in set (0.00 sec)

mysql> select 6 << 2, 6 >> 2;
+--------+--------+
| 6 << 2 | 6 >> 2 |
+--------+--------+
|     24 |      1 |
+--------+--------+
1 row in set (0.00 sec)
